## Force password reset on login

### Usage

- Clone the repo to /tmp
  ```bash
  git clone https://github.com/k8-proxy/GW-proxy /tmp/GW-proxy
  ```

- Run the script
  ```bash
  git clone https://github.com/k8-proxy/GW-proxy /tmp/GW-proxy
  /tmp/GW-proxy/automation/scripts/pw-reset-onlogin/pw-reset-onlogin.sh
  ```

- Logout `logout`
